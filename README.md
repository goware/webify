Webify
======

Simply serve your current directory over HTTP.

## Install / Usage

```shell
$ go install github.com/goware/webify@latest
```

```shell
$ cd somepath/
$ webify
================================================================================
Serving:  /Users/peter/somepath
URL:      http://0.0.0.0:3000
================================================================================
```

run `webify -h` for help
